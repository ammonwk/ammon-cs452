# CS 452 Final Project - Plaibook AI Outbound Agent

An AI-powered outbound messaging and conversation management platform for CS 452 (Data Modeling & NoSQL).

## Project Overview

**Plaibook** automates customer engagement through AI-driven SMS campaigns while providing real-time human oversight. The system processes call recordings with AI for quality scoring and coaching insights.

## Documents

- [Initial Design](./INITIAL_DESIGN.md) - Project goals, ERD, system architecture
- [Schema](./schema.sql) - MySQL DDL representation of the data model

## Tech Stack

- **Frontend:** React 19, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express, tRPC, MongoDB
- **AI:** OpenRouter (Claude, GPT-4, Gemini)
- **Infrastructure:** AWS (ECS, Lambda, S3, SQS)
- **Real-time:** Socket.IO, Redis
